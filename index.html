Project

Org Chart App

Sponsored by Ultimate Software Group, Inc

  • Building a microservice for Ultimate Software Inc to expand their services
  • Developed a web app using Angular as front-end, NestJs as Back-end Server, and MongoDB
  • Let employee see their company org char, Manager is able to transfer their employee to other department which will shown in the org chart
  • Using JWT for authentication(refresh token, access token)
  • Using docker to containerize the front-end and the back-end server also the configure local mongoDB with replica sets
  • Transactions make sure the data integrity
  • Deployed front-end server use GitHub Page
  • Deployed back-end server use Heroku
  • Deployed database server use Mongo Atlas
